#### **文字・語彙**

**問題1** ______ のことばの読み方として最もよいものを、1・2・3・4から一つえらびなさい。

1. 荷物は、もう**届いて**います。  

   1. あいて　2. ほどいて　3. ついて　4. とどいて

2. **適度**な運動は、体によい。  

   1. てきと　2. てきとう　3. てきど　4. てきどう

3. 部長は今、**大阪**へ出張しています。  

   1. しゅちよう　2. しゅっちょう　3. でちょう　4. でっちょう

4. 両親が子どもたちを**養う**。  

   1. さそう　2. やしなう　3. ととのう　4. うたがう

5. **勇気**がなくて、好きな人に話しかけられない。  

   1. よき　2. ようき　3. ゆき　4. ゆうき

6. 空港で日本円をユーロに**両替**した。  

   1. りょかえ　2. りょがえ　3. りょうかえ　4. りょうがえ

7. 遠くに白い**煙**が見えます。  

   1. くも　2. きり　3. けむり　4. はい

8. 家の前で車が**停止**した。  

   1. ていし　2. てんし　3. ていじ　4. てんじ

**問題2** ______ のことばを漢字で書くとき、最もよいものを1・2・3・4から一つえらびなさい。

12. ホストファミリーが私をあたたかく**むかえて**くれた。  

    1. 拝えて　2. 待えて　3. 迎えて　4. 歓えて
