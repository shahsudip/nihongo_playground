**問題1** 次の文の（　　）に入れるのに最もよいものを、一つえらびなさい。

1. 台風が来て、昨日は沖縄では大雨だった（　　）。  
   1. のは　2. とを　3. のを　4. とか

2. 妻「あなたの体のために、明日からたばこを吸うのをやめてちょうだい。」  
   夫「（　　）厳しいことを言わないでよ。」  
   1. こんなに　2. そんなに　3. あんなに　4. どんなに

3. 私は昨日から**風邪**（　　）、熱が少しある。  
   1. っぽい　2. ほど　3. げ　4. 気味

4. 近所へ散歩に行った（　　）、スーパーで牛乳を買ってきました。  
   1. かたがた　2. ついでに　3. かわりに　4. ままに

5. 学校から家へ帰る（　　）彼女に会ったので、道で少し話しました。  
   1. 途中まで　2. 途中へ　3. 途中で　4. 途中から

6. この町は、駅を（　　）発展してきた。  
   1. 中心とともに　2. 中心として　3. 中心において　4. 中心にかけて

7. 毎朝私は、ベランダの花に水を（　　）。  
   1. くれます　2. もらいます　3. やります　4. さしあげます

8. 彼女に会うのは、卒業以来3年（　　）。  
   1. よりだ　2. ながらだ　3. からだ　4. ぶりだ

9. 弟は先生に何と（　　）。  
   1. お話しなさいましたか　2. 話されましたか  
   3. おっしゃいましたか　4. 申しましたか

10. あの子の、小さい子ども（　　）、とてもかわいい。  
    1. にしては　2. のあまり　3. であるからには　4. からして

11. 男の学生「あ、ペンを忘れちゃった。」  
    女の学生「私（　　）、これを使ってよ。」  
    1. のもよければ　2. のでよければ　3. がよいから　4. でよいから

12. あまり時間がないから、食事を（　　）、簡単なものにすることつもりだ。  
    1. するとしては　2. するとしようと　3. するとしても　4. するとしようなら

13. 私はアイスクリームが大好きなんで、お店で見かけると（　　）。  
    1. 買おうとしています　2. 買おうとはしません  
    3. 買わないでいられます　4. 買わないではいられません

**問題3** つぎの文章は、留学生の張さんが書いた作文です。［　19　］から［　23　］の中に入る最もよいものを、1・2・3・4から一つえらびなさい。

**富士山のこと**

張 拓

土曜、日曜日に、日本に来て初めて、富士山に行きました。日本に来る前は、日本のイメージを代表するものは富士山でした。富士山に［　19　］、前の日の夜はうれしくて、よく眠れませんでした。

私が友だちと一緒に参加したツアーは、朝早くバスで出発し、まずバスで五合目まで行き、その後、ガイドさんに従って、八合目までのぼり、山小屋にとまって、次の日の朝、頂上を目指すというものです。とても天気がよかったので、頂上で「ご来光」と呼ばれる朝日を［　20　］。本当にきれいでした。

ただ、残念だったのは、ごみがたくさん捨てられていたことです。［　21　］、友だちと下りるときにごみを拾うことにしました。もっていた袋に、見つけたごみを入れていくと、五合目に戻るころには、ごみ袋がいっぱいでした。遠くから見るとあんなに［　22-a　］富士山も、実際はこんなに［　22-b　］ことがわかって、悲しい気持ちになりました。ガイドさんも、富士山のごみは問題になっている、と言っていました。

帰ってきてから、パソコンで［　23　］、山を登りながらごみを拾うツアーもあるようです。富士山をきれいにするために、今度はこのようなツアーに参加したいと思いました。


**19**  
1. 登れると思うと  
2. 登れたと思うと  
3. 登れたと思ったら  
4. 登れと思ったら

**20**  
1. 見られそうでした  
2. 見ることでした  
3. 見ることができました  
4. 見たいと思いました

**21**  
1. ところで  
2. そこで  
3. ところが  
4. それが

**22**  
1. a. きたない　/　b. よごれている  
2. a. きたない　/　b. よごれていない  
3. a. きれいな　/　b. よごれていない  
4. a. きれいな　/　b. よごれている

**23**  
1. 調べるものの  
2. 調べようと  
3. 調べるにしたら  
4. 調べたところ
